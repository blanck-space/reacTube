# reacTube

Simple ReactJS based YouTube instant search client.


####Clone it!#####

```
	> git clone git@github.com:StephenGrider/ReduxSimpleStarter.git
	> cd ReduxSimpleStarter
	> npm install
	> npm start
```

####Test it!#####

```
	> npm install
	> npm start
```
(based on ReduxSimpleStarter from StephenGrider)
